 │     91                                                                                                                                                                                                                                                                          │

 │     97 \end{itemize}                                                                                                                                                                                                                                                            │
 │     98                                                                                                                                                                                                                                                                          │
 │     99 Der Miner sucht nach einem gelösten Zustand $S_f$ und der Zugsequenz $M$, so dass:                                                                                                                                                                                       │
 │    100 $$H(\text{block\_header} \parallel \text{solution\_hash}(M)) < D$$                                                                                                                                                                                                       │
 │    101                                                                                                                                                                                                                                                                          │

 │    107 \item Der Lösungshash das Schwierigkeitsziel erfüllt                                                                                                                                                                                                                     │
 │    108 \item Der Anfangszustand mit dem Block-Header konsistent ist                                                                                                                                                                                                             │
